# Context Engine

## Objective

Understand organizational context before generating responses.

---

## Context Sources

- Department
- User Role
- Historical Data
- Project Status
- Business Rules

---

## Example

CEO asks:

Which projects are risky?

Context Engine evaluates:

- Budget
- Schedule
- Team Capacity
- Vendor Performance

before generating answer.

---

## Benefits

- Personalized Responses
- Better Accuracy
- Reduced Hallucination Risk
